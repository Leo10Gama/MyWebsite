Code Projects

  • L30-Series Discord Bots

    These Discord bots each do somewhat similar tasks, with some differences. The first model, L30-Gamma, uses JavaScript, while the second model, L30-Pi, uses Python. Each makes use of similar API provided by Discord to interact with the server interface, as well as other API such as PokeAPI or BeautifulSoup to perform supplementary tasks.

    Each bot does its own thing, but their primary purpose is to show pictures of cats. Images are shown either at random, or a random image of a specific cat can be seen if that cat's name is entered following the command. In addition, the L30-Pi model is capable of interacting with websites like Numista, NinSheetMusic, and KHInsider to get more information oncoins, sheet music, or video game soundtracks respectively.
